2012.12.10 GitHub 와 이클립스 연동 방법 

1. gitHub에 프로젝트 등록 워크스페이스를 몽땅 등록할 필요는 없음. 

2. 새로운 PC에서 gitHub를 설치하고 Clone한다. 
	-> 폴더는 자기 하고 싶은데로 하면 된다. 
	-> 나는 d:\GitHub로 설정 

3. Clone후에 eclipse를 오픈 

4. 프로젝트를 신규로 생성한다. 
  -> 프로젝트 신규 생성 시 
  -> 프로젝트 명 아래의 project Location 그룹박스에서 
     "use default Location을 uncheck하고, 
      Location을 GitHub안의 프로젝트 폴더로 설정한다. 

5. 변경된 내용은 GitHub가 알아서 확인하고 Sync여부를 확인한다. 


앞으로 잘 써먹을 수 있을거 같다. 

잘살아 보세...대박나라.
